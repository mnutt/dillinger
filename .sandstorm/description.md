# Dillinger

Dillinger is a focused Markdown editor with a live, side-by-side preview.

Each Sandstorm grain contains one independently shareable document. Changes are saved automatically to the grain, and the **Publish** button renders the document as a static website using Sandstorm web publishing.

Features include Monaco-powered editing, Markdown and HTML import, Markdown/HTML/PDF export, image embedding, Vim and Emacs keybindings, and distraction-free zen mode.
